# Markdown Q&A

## 如何在表格中输入竖线（|）？

*   直接使用 `&#124;` 代替竖线。

*   对于在代码语句中的竖线，使用 <code>&#60;code&#62;&#60;/code&#62;</code> 替换 <code>`</code>。

参考：[How to escape a pipe char in a code statement in a markdown table?
